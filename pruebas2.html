Estructura básica y Metadatos

<!DOCTYPE>Define el tipo de documento y la versión

<html>Raiz de un documento html

<head>Contienemetadatos/informacioón sobre el documento

<title>Titulo del documento

<meta>Metadatos sobre el HTML, como codificación de caracteres, autor del documento, etc

<link>Enlace a hojas de estilo externas (CSS)

<style>Define estilos CSS internos para el documento

Secciones y Agrupación de Contenido

<body>Contiene el contenido del documento que se muestra en el navegador

<header>Representa un grupo de ayudas introductorias o de navegación

<footer>Representa el pie de página de un documento o sección

<nav>Sección del documento destinada a la navegación

<section>Define una sección dentro de un documento

<article>Contenido independiente y autocontenido

<aside>Contenido menos relacionado con el contenido principal

<div>Contenedor genérico para contenido de flujo, sin significado semántico

<span>Contenedor genérico para contenido en línea, sin significado semánticol

Texto y Enlaces

<h1, h2, ..... h6>Encabezados de sección de diferentes niveles

<p>Párrafo

<a>Enlace (hipervínculo) a otra página o recurso

<strong>Texto de importancia mayor

<em>Texto enfatizado (generalmente itálico)

<br>Salto de línea

<blockquote>Citas de secciones externas
